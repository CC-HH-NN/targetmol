# Example Results

This directory contains sanitized excerpts from real TargetMol runs. They are intended for readers who want to inspect the shape of intermediate outputs without downloading model weights or reproducing the full server environment.

Included cases:

- `screen_only/`: uploaded candidate SMILES screened against PDB ID `6JX0`.
- `sbdd_pdb_id/`: PDB ID `6JX0` routed through structure-based generation and screening.
- `ligand_based/`: natural-language ligand-agent refinement with search, LLM grounding, embedding-ranked evidence, and iterative candidate updates.

File types:

- `summary.txt`: short route-level run summary.
- `agent_report.txt`: natural-language report generated by TargetMol.
- `run_metadata_public.json`: sanitized route metadata and step names.
- `final_report_public.json`: compact screening metrics and top candidates where screening was run.
- `top_candidates.csv`: compact candidate ranking table where screening was run.
- `iterative_ligand_refinement_public.json`: compact iterative refinement trace for the ligand-agent case.
- `final_candidates.smi`: final ligand-agent candidates.

These results are computational hit-discovery examples only. They are not experimental validation results and should not be interpreted as confirmed drug candidates.
